Friendly Fixtures, 2018-2019, DMU Independent Web Application Development Project

Friendly Fixtures was a responsive web application I chose to develop for my University dissertation. This application demanded the process of a professional project, including: project management, research, documentation, use cases, and testing, proving my ability to independently manage a project. Weekly meetings were undertaken to track progress, adopting a scrum methodology, recording what changes were made in the project and determining whether future changes were advisable. The base languages included: HTML, CSS, JavaScript. This controlled the visuals of the application and any dynamic elements such as the responsive mobile menu. The application was supported by a MySQL database backend hosted locally using XAMPP to run Apache & MySQL. The database would securely store user account information by hashing passwords with the BCRYPT function, when required they would be dehashed to log the user into the application.

Wellington Hotel Booking System, 2017-2018, NCN Visual Studio Team Development Project

This project was achieved during my HND at New College Nottingham. The project was team based & required us to follow a professional project life cycle to produce all the required documents and software for completion. We were handed a client brief in which we had to provide a solution for a hotel still using a paper-based documentation system. I first assessed the team and assigned roles based on strengths, this ensured all team members were comfortable and efficient when producing documents. Weekly meetings were undertaken to discuss current tasks and progress and avoid the repeating of already completed processes. After thoroughly researching the task we began developing: communication plans, quality assurance plans, project contracts, & database designs. I then distributed work evenly on Visual Studio, assigning a section of the application to each team member which was coded in C#, with a combination of PHP & MySQL to support the datbase functionality.
